# 🏭 Chemical Plant Digital Twin

A real-time **Chemical Plant Digital Twin** developed using **Python, Streamlit, SQLite, Pandas, and Matplotlib**. The application simulates industrial plant operations, visualizes process data, performs engineering calculations, stores historical data, and provides an interactive dashboard for process monitoring.

## 🌐 Live Demo

🔗 https://chemical-plant-dashboard-priyansh.streamlit.app/

---

## 📌 Project Overview

This project simulates the operation of a chemical plant through a digital twin dashboard. It generates realistic process sensor data, monitors equipment performance, visualizes trends, stores historical process data, and provides engineering insights using interactive dashboards.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Priyansh58/Chemical-Plant-Dashboard.git
```

Move into the project directory

```bash
cd Chemical-Plant-Dashboard
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
