# 🧪 Chemical Plant Digital Twin

A Streamlit-based digital twin dashboard for a chemical plant.

## Features

- 📊 Live Process Dashboard
- 🌡 Temperature Monitoring
- ⚙ Pressure Monitoring
- 💧 Flow Rate Monitoring
- ⚡ Efficiency Calculation
- 🔥 Heat Duty Calculation
- 💰 Operating Cost Estimation
- 🚨 Smart Alarm System
- 📈 Interactive Charts
- 📋 Sensor Data Table
- 📥 CSV Download
- 📊 Statistical Summary

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
